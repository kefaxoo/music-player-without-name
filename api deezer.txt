deezer api:
Каждые 5 секунд можно запросить 50 запросов (лимит api)

Доступны несколько вариантов "получения" данных:
- .json = json
- .xml = xml
- .php = json + xml
Пример:
https://api.deezer.com/album/302127&output=xml

deezer api:

api.deezer.com/chart/0/tracks - какой-то чарт (есть 30-ти секундные превью)
api.deezer.com/search/album?q= - поиск по альбомам (есть 30-ти секундные превью)
api.deezer.com/search/track?q= - поиск по трекам
api.deezer.com/search/artist?q= - поиск по исполнителям

"у дизера есть лимит api, который состовляет 50 запросов каждые 5 секунд."

QUALITY:
128 - MP3 128 kbps (in 2022 works only 128 kbps downloading)
320 - MP3 320 kbps
1411 - Flac 1411 kbps

dz.loaderapp.info/deezer/QUALITY/url

QUALITY - цифрами
id - цифры трека из api

чтобы скачать трек из поиска треков:
1) api.deezer.com/search/track?q=НАЗВАНИЕ ТРЕКА
2) берём "link":"http:\/\/www.deezer.com\/track\/id" (это url) нужного трека
3) убираем из url "http:\/\/www." и лишний слэш везде
4) полученный url вставляем в dz.loader.app.info/deezer/QUALITY/url
5) скачиваем трек)

чтобы скачать трек из поиска альбомов:
1) api.deezer.com/search/album?q=НАЗВАНИЕ АЛЬБОМА
2) берём "tracklist":"http:\/\/api.deezer.com\/album\/108587042\/tracks" (это url) нужного альбома
3) убираем из url "http:\/\/www." и лишний слэш везде
4) открываем url трэклиста
5) чтобы скачать трек, идём в часть "чтобы скачать трек из поиска треков:" к пункту 2 и следуем плана

чтобы скачать трек из поиска исполнителей:
1) api.deezer.com/search/artist?q=НАЗВАНИЕ ИСПОЛНИТЕЛЯ
2) берём "tracklist":"http:\/\/api.deezer.com\/artist\/11518551\/top?limit=50" (это url) нужн
3) открываем url трэклиста и ищем что нам надо (популярные (первые 50) треки (4) или альбом (5)
4) чтобы скачать трек, идём в часть "чтобы скачать трек из поиска треков:" к пункту 2 и следуем плана
5) чтобы скачать альбом, идём в часть "чтоб скачать трек из поиска альбомов:" к пункту 2 и следуем плана

чтобы скачать трек из чарта:
1) api.deezer.com/chart/0/tracks
2) чтобы скачать трек, идём в часть "чтобы скачать трек из поиска треков:" к пункту 2 и следуем плана
